
<< PRÉPARATION >>
 ( ) COMMUNIQUER AU CLIENT LES TERMES DE CETTE INSPECTION.
 ( ) COORDONNER AVEC UN AUTRE MÉTIER SI NÉCESSAIRE.
 ( ) INSTALLER "AVIS DE MAINTENANCE" DANS LA SALLE DE CONTRÔLE
     ET REMPLIR LA FICHE DE COMMUNICATION.
 ( ) UN PERMIS D'ESPACE CLOS PEUT ÊTRE NÉCESSAIRE POUR FAIRE
     LES OPÉRATIONS 2, 4, 5 ET 6.

<< CONSOMMABLES >>
 - LUBRIFIANT SEC TFL-50:   #LCPMC79904-2

<< VÉRIFICATIONS AVEC L'UNITÉ EN FONCTION >>
 ( ) VÉRIFIER LES BRUITS ET VIBRATIONS INHABITUELS.
     AU BESOINS, UTILISER UN STÉTOSCOPE.
 ( ) VÉRIFIER LA COURSE DES VOLETS.
 ( ) VÉRIFIER LES INTERRUPTEURS DE FIN DE COURSE.


<< SÉCURITÉ >>
 ( ) APPLIQUER TOUTES LES PROCÉDURES DE SÉCURITÉ RELATIVES AUX
     TRAVAUX; RÈGLES CARDINALES, CADENASSAGE, PERMIS DIVERS,
     EPI, ETC.
 ( ) VALIDER LA FICHE DE CADENASSAGE; PROPRE, LISIBLE ET À JOUR.

<< TÂCHES >>
 ( )  1 - INSPECTER LES GRILLAGES D'ENTRÉES D'AIR.
 ( )  2 - AVEC L'AIDE DE L'ÉLECTRICIEN, FAIRE FONCTIONNER
          L'ACTUATEUR DES VOLETS POUR VÉRIFIER QU'IL N'Y A PAS
          DE FUITES D'AIR ET QU'ILS FONCTIONNENT BIEN.
 ( )  3 - SI NÉCESSAIRE, APPLIQUER UNE LÉGÈRE COUCHE DE
          LUBRIFIANT SEC #LCPMC79904-2 SUR LES PIÈCES
          MOBILES DU MÉCANISME DES VOLETS.
 ( )  4 - VÉRIFIER LA PROPRETÉ DES CONDUITS.
 ( )  5 - VÉRIFIER VISUELLEMENT S'IL Y A UN DÉPÔT DE ROUILLE
          SUR LES JOINTS OU AUTRES ENDROITS SUSCEPTIBLES D'EN
          AVOIR.
 ( )  6 - S'ASSURER QUE L'AFFICHE INDIQUANT L'ESPACE CLOS ET
          SON NUMÉRO EST VISIBLE ET CONFORME.
 ( )  7 - VÉRIFIER L'IDENTIFICATION SUR L'ÉQUIPEMENT
          (MINIMUM 2 FACES) ET LA REFAIRE AU BESOIN.
          - NOTE: LES CONDUITS SUR LE TOIT DOIVENT ÊTRES
                  IDENTIFIÉS (PRIMAIRE ET SECONDAIRE)
                  AINSI QUE LE NO. D'ÉQUIPEMENT POUR CHACUN.

( ) NE PAS METTRE EN FONCTION SI ÉQUIPEMENT N'EST PAS SÉCURITAIRE.
( ) LORSQUE LE TRAVAIL EST TERMINÉ: DÉCADENASSER, COMPLÉTER LE
    FORMULAIRE DE CADENASSAGE PWC 1834 ET LE BROCHER AU PM.
( ) COMPLÉTER LA FICHE DE COMMUNICATION.
( ) S'ASSURER QUE LES LIEUX DE TRAVAIL SOIENT PROPRES EN QUITTANT.
( ) AVISER L'OPÉRATEUR OU LE SUPERVISEUR DE LA FIN DES TRAVAUX.
