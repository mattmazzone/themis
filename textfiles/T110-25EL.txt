
<< PRÉPARATION >>
 ( ) COMMUNIQUER AU CLIENT LES TERMES DE CETTE INSPECTION.
 ( ) COORDONNER AVEC UN AUTRE MÉTIER SI NÉCESSAIRE.
 ( ) INSTALLER "AVIS DE MAINTENANCE" DANS LA SALLE DE CONTRÔLE ET REMPLIR LA FICHE DE COMMUNICATION.

<< DOCUMENTS RÉFÉRENCES >>
 - PROCÉDURE : PW-SLP 22.14, FICHE DE VÉRIFICATION ÉLECTRIQUE.

<< SÉCURITÉ >>
 ( ) APPLIQUER TOUTES LES PROCÉDURES DE SÉCURITÉ RELATIVES AUX
     TRAVAUX; RÈGLES CARDINALES, CADENASSAGE, PERMIS DIVERS,EPI, ETC.
 ( ) VALIDER LA FICHE DE CADENASSAGE; PROPRE, LISIBLE ET À JOUR.

<< TÂCHES >>
 - NOTE : TOUTES LES BOÎTES ÉLECTRIQUE DOIVENT ÊTRE FERMÉES
	     SUIVANT LA PROCÉDURE PW-SLP 22.14.

 ( )  1 - ASSISTER LE MÉCANICIEN POUR SON ENTRETIEN DES VOLETS:
          - FAIRE FONCTIONNER L'ACTUATEUR DES VOLETS.
 ( )  2 - VÉRIFIER LES LIMIT SWITCHS DES VOLETS.
 ( )  3 - VÉRIFIER L'IDENTIFICATION SUR L'ÉQUIPEMENT (MINIMUM 2 FACES) ET LA REFAIRE AU BESOIN.

( ) NE PAS METTRE EN FONCTION SI ÉQUIPEMENT N'EST PAS SÉCURITAIRE.
( ) LORSQUE LE TRAVAIL EST TERMINÉ: DÉCADENASSER, COMPLÉTER LE
    FORMULAIRE DE CADENASSAGE PWC 1834 ET LE BROCHER AU PM.
( ) COMPLÉTER LA FICHE DE COMMUNICATION.
( ) S'ASSURER QUE LES LIEUX DE TRAVAIL SOIENT PROPRES EN QUITTANT.
( ) AVISER L'OPÉRATEUR OU LE SUPERVISEUR DE LA FIN DES TRAVAUX.
