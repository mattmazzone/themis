Application .NET

Cette section de la documentation explique comment le PM.html interagit avec les applications EmailClient et FileClient (.exe)

C'est quoi un programme .NET?

Le .NET Framework est un cadriciel pouvant être utilisé par un système d'exploitation Microsoft Windows. Il a pour but de faciliter la tâche des développeurs en introduisant des facilités pour le développement, le déploiement et la maintenance d'application Windows.

Le C# a été choisi pour cette application parce que le API de Outlook doit être utilisé pour envoyer un courriel. C# et Outlook sont des produits de Microsoft et c'est facile de faire le lien entre les deux.

Fonctionnement de EmailClient

Lorsque l'utilisateur appuie sur le bouton Replanifier, du code JavaScript est exécuté. Si toutes les conditions sont respectées, la fonction JavaScript va permettre l'enregistrement du PDF. L'utilisateur va devoir sélectionner le dossier téléchargement et confirmer l'enregistrement. Quand la fenêtre de sauvegarde ferme, une autre fonction JavaScript va être appelée. Cette fonction est unique au bouton Replanifier. La fonction, fera débuter le téléchargement du fichier EmailClient.exe. C'est possible que le navigateur affiche un avertissement de sécurité la première fois que ce téléchargement est effectué. C'est moi (Matteo) qui a écrit le code pour cette application et elle est sécuritaire.

L'utilisateur devra ensuite lancer cette application. Le script serra exécuté en arrière-plan et un courriel va être envoyé aux destinataires dans le fichier outlook_details.txt (O:\TFM_GESTION\PM\1_TEMPLATES\WebCompanion). Il y aura une confirmation apres une dizaine de secondes que le courriel a été envoyé ou si il y a eu une erreur.

Le fichier EmailClient.exe qui est télécharger après que le bouton "À Replanifier" est sélectionné est téléchargé à partir du lien: "O:\TFM_GESTION\PM\1_TEMPLATES\WebCompanion\EmailClient.exe"
Si ce fichier est modifié, il est important de garder le même nom.

Fonctionnement de FileClient

Le fonctionnement de FileClient est identique à EmailClient. En plus d'envoyer un courriel aux destinataires dans le fichier outlook_details.txt, une copie du pdf est enregistré sur le drive O:

Le fichier FileClient.exe qui est télécharger après que le bouton "Soumettre" est sélectionné est téléchargé à partir du lien: "O:\TFM_GESTION\PM\1_TEMPLATES\WebCompanion\FileClient.exe"
Si ce fichier est modifié, il est important de garder le même nom.
