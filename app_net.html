Application .NET

Cette section de la documentation explique comment le PM.html interagit avec les applications EmailClient et FileClient (.exe)

C'est quoi un programme .NET?

Le .NET Framework est un cadriciel pouvant être utilisé par un système d'exploitation Microsoft Windows. Il a pour but de faciliter la tâche des développeurs en introduisant des facilités pour le développement, le déploiement et la maintenance d'application Windows.
Il est possible d'utiliser plusieurs languages de programmation pour créer des applications basées sur le .NET Framework. Le C# a été choisi pour cette application parce que l'outil de conversion des PM est en C++ et les deux langues de programmation ont une syntaxe similaire.

Fonctionnement de EmailClient

Lorsque l'utilisateur appuie sur le bouton Replanifier, du code JavaScript est exécuté. Si toutes les conditions sont respectées, la fonction JavaScript va permettre l'enregistrement du PDF. L'utilisateur va devoir sélectionner le dossier téléchargement et confirmer l'enregistrement. Quand la fenêtre de sauvegarde ferme, une autre fonction JavaScript va être appelée. Cette fonction est unique au bouton Replanifier. La fonction, fera débuter le téléchargemnt du fichier EmailClient.exe
L'utilisateur devra ensuite lancer cette application.
